How to Install YOS

  1. Download the ISO: Visit the Download Section and get the latest YOS version.
  2. Create Bootable Media: Use Rufus or Balena Etcher to flash the ISO onto a USB or SD card.
  3. Boot Your Device: Insert the bootable media into your device, restart, and select the media to boot from.
  4. Install YOS: Follow the on-screen instructions provided by the YOS installer.
